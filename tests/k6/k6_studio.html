                    ⬤ 실시간 테스트 콘솔 모니터
                    
                

                

            


            
            

                
4. k6 스크립트 실행 가이드

                
                

                    생성된 코드를 컴퓨터에 파일로 저장한 뒤, 터미널(PowerShell 또는 bash)을 열고 아래 명령어들을 실행하면 부하 테스트가 작동합니다.
                


                
Step 0. 사전 토큰 발급 (최초 1회 또는 유저 추가 시)

                
테스트 전 반드시 실행하여 tokens.json을 생성합니다. --count는 최대 VU 수 이상으로 설정하세요.

                
node generate_tokens.js --url http://<API_BASE_URL>/api/v1 --count 1000

                
💡 서버 과부하 방지: --batch 30 --delay 500 옵션으로 속도 조절 가능


                
Step 1. k6 부하 테스트 도구 설치하기

                
Windows 환경 (Chocolatey 사용 시):

                
choco install k6

                
Mac 환경 (Homebrew 사용 시):

                
brew install k6


                
Step 2. 부하 테스트 실행 (콘솔 로그 확인)

                
저장한 스크립트가 있는 폴더로 이동한 후 실행합니다.

                
k6 run spike_test.js


                
Step 3. 그라파나 대시보드와 실시간 연동하여 실행 (강력 추천)

                

                    테스트 도중 모니터링 대시보드에 실시간 그래프를 나타나게 하려면 아래 명령어로 프로메테우스 출력을 지정해야 합니다.
                

                
k6 run --out experimental-prometheus-trends spike_test.js

                💡 그라파나 화면이 빈 칸으로 보이나요?
