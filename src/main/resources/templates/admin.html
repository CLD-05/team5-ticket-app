💡 대기열 리셋 기능이란? 실시간 대규모 접속 시, 사용자들은 먼저 Redis Sorted Set 구조의 대기열에 진입하고 차례대로 입장 권한 토큰을 발급받아 좌석 선택 화면에 도달합니다.
  • 공연별 리셋: 특정 공연의 대기 큐 및 이미 발급된 입장 토큰만 선별적으로 삭제하여 진입로를 복구합니다.
  • 전체 강제 리셋: Redis 내의 모든 대기열 관련 데이터(`queue:*` 와 토큰 키)를 삭제하여 무임승차를 차단하고 초기 클린 상태로 원복합니다.
